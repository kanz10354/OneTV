# OneTV

OneTV is a modern streaming application for macOS with advanced features including SmartWatch (multi-source streaming), next episode autoplay, and extensive addon support.

## Installation

1. Download the appropriate DMG for your Mac architecture
2. Double-click the DMG file
3. Drag OneTV to your Applications folder
4. Launch OneTV from Applications or Launchpad
